# 🤖 JETTY V5 — Your AI Second Brain (Tools Edition)

A voice assistant that lives on top of YOUR notes and YOUR tools, running locally on your machine.

Start here: open **GET-STARTED.md** (Mac) or **GET-STARTED-WINDOWS.md** (Windows).

## ✨ What V5 can do

- 🔌 **Tool Armory**: connect Canva, Notion, GitHub and more with one click, add ANY service that publishes an MCP server URL, and optionally bridge everyday apps (email, calendar) through a service like Zapier. "Check my email", "what's on my calendar tomorrow", "draft a reply". Every answer shows a receipt of which tool actually ran
- 🗣 **Real conversation**: hands free wake word, instant interrupt (just start talking over him), "get back to me when it's done" quiet mode, read it myself screen mode
- 🧠 **Brain hot swap by voice**: opus, sonnet, haiku, fable, and with OpenRouter one key unlocks grok, gpt, gemini, deepseek and more
- 🌐 **Live web research**: "research the latest AI news" with sources on screen
- 🗂 **Living notes**: "remember that..." and "create a project for..." write real markdown files and grow the 3D graph on the spot
- ⏰ **Tasks, reminders and briefings**: "remind me tomorrow at 9", "good morning"
- 🌍 **16 languages**: "speak Spanish" and the whole assistant follows
- 🎭 **Personas and dials**: "be less funny", "go full GLaDOS"
- 🚀 **Missions** (needs the Claude Code CLI): a 3 agent research fleet, or "build me an app" that ships a working page

## 🔑 Keys

Only ONE key is required: Anthropic (console.anthropic.com), pasted into config.json. Everything else is optional and documented in config.example.json.

## 🔒 Safety design

Drafts never send. Purchases never happen. Jetty asks before anything irreversible, and answers about your data always come from real tool results with a visible receipt.

Built with Claude Code. Enjoy! 🚀
